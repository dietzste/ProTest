ProTest ist ein Tool zur Unterstützung von Programmiertestungen.

Installation
1. Download Protest_v2.0.
 - über Clone or Download > Download ZIP 	ODER
 - https://github.com/dietzste/ProTest/archive/master.zip 
 - anschließend entpacken
2. Download Capture2Text (v4.6.2_64bit) from: https://sourceforge.net/projects/capture2text/files/latest/download
3. Den Ordner Capture2Text in den Ordner Protest_v2.0 verschieben
